P1. Conexiones de redes y gestión en Linux

Esta práctica consiste en la configuración y análisis de una red local entre dos máquinas Linux (servidor y cliente) dentro de la red 192.168.50.0/24. Se realizan comprobaciones de conectividad, servicios y configuración del sistema.

---

1. Interfaces de red

Se usa ip a para ver interfaces de red, IP y MAC.

ip a

Se identifica la interfaz activa y su dirección IP en la red local.

---

2. Configuración de red

Se usa ip addr y ip route para ver la red configurada.

ip addr
ip route

Se muestra la red 192.168.50.0/24 y la interfaz usada.

---

3. Hostname

Se consulta y cambia el nombre del equipo.

hostname
sudo hostnamectl set-hostname srv01

El hostname identifica la máquina dentro de la red.

---

4. Netplan

Configuración de IP estática mediante Netplan.

sudo nano /etc/netplan/01-netcfg.yaml
sudo netplan apply

Permite asignar IP fija a la máquina.

---

5. Ping

Comprobación de conectividad entre máquinas.

ping 192.168.50.10

Si funciona, se reciben respuestas con tiempo de latencia.

---

6. Hosts

Resolución de nombres mediante /etc/hosts.

sudo nano /etc/hosts
ping srv01

Permite usar nombres en lugar de IPs.

---

7. Rutas

ip route

Muestra la tabla de rutas del sistema.

---

8. Puertos

ss -tuln

Se muestran los puertos abiertos en el sistema.

---

9. SSH

sudo apt install openssh-server
sudo systemctl status ssh

Permite acceso remoto seguro al servidor.

---

10. Acceso remoto

ssh usuario@192.168.50.10
whoami
hostname

Conexión remota al servidor Linux.

---

11. Interfaces estado

ip link

Se comprueba el estado UP/DOWN de las interfaces.

---

12. ARP

ip neigh

Relación entre IP y MAC de los equipos.

---

13. SCP

scp prueba.txt usuario@192.168.50.10:/home/usuario

Transferencia de archivos entre cliente y servidor.

---

14. Control SSH

sudo systemctl stop ssh
sudo systemctl start ssh

Se comprueba el funcionamiento del servicio SSH.

---

15. Reinicio

ip a
hostname
systemctl status ssh

Se verifica que la configuración se mantiene tras reiniciar.
