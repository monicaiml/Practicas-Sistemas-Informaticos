1. Interfaces de red

Comando:

ip a

Explicación: muestra interfaces, IP y MAC del sistema.

Captura:

2. Configuración de red

Comando:

ip addr
ip route

Explicación: muestra configuración IP y rutas del sistema.

3. Hostname

Comando:

hostnamectl set-hostname srv01

El hostname identifica el equipo en la red.

4. Netplan

sudo nano /etc/netplan/01-netcfg.yaml
sudo netplan apply

Configura IP estática del sistema.

5. Ping

ping 192.168.50.10

Comprueba conectividad entre máquinas.

6. Hosts

sudo nano /etc/hosts

Resuelve nombres localmente sin DNS.

7. Rutas

ip route

Muestra rutas y gateway del sistema.

8. Puertos

ss -tuln

Lista puertos en escucha.

9. SSH

sudo apt install openssh-server
systemctl status ssh

Servicio de acceso remoto seguro.

10. Acceso remoto

ssh usuario@192.168.50.10
whoami
hostname

Conexión remota al servidor.

11. Interfaces estado

ip link

Muestra estado UP/DOWN de interfaces.

12. ARP

ip neigh

Relación IP ↔ MAC en la red local.

13. SCP

scp prueba.txt usuario@192.168.50.10:/home/usuario

Transferencia de archivos por SSH.

14. SSH control

sudo systemctl stop ssh
sudo systemctl start ssh

Control del servicio SSH.

15. Persistencia

ip a
hostname
systemctl status ssh

La configuración se mantiene tras reiniciar.
